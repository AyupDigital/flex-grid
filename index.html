Installation

Install using NPM.

Usage

You have two different options for using FlexiGrid. You may include the framework alone and create your column classes yourself, or you can use the pre-defined breakpoints and generated class names.

Base

This includes only the Sass mixins that power flexigrid, with no specific class names.

With Classes

This includes the Sass mixins that power flexigrid, as well as responsive column classes.

Setup

flexigrid is as agnostic as possible in regards to your layout and breakpoints. You can use the semantic grid mixin to generate rows and columns for your layout and components, or you can generate responsive class names based on the breakpoints used in your project.

Settings

The flexigrid library has variables that help configure the base library and library with classes. You may override these in your own project if you wish to customize your grid layout.
